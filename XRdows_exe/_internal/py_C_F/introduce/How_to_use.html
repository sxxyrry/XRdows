How to use:

C_T is Command table (Once entered, a command table is output)

_EXIT is exit (It exits after typing)

P_W is plane war game (Entering it will open the plane war game)

_E_L is Edition logs (After input, the version record is output)

_I_H_ is introduce html (After entering, the introduction page opens)

print() is print (After typing, it will allow you to continue typing text so that you can print)

M_P is music player (Once typed, it will allow you to continue typing text (path) so that the music can be played)

R_py_F is to run the python file (after entering, enter the python file path)

R_exe_F is to run the executable (after entering, enter the executable path)

M_P_D is the music player by default (the music files are in the "music_file" folder)

But, of course, there are some specific features that Developer Mode doesn't offer
